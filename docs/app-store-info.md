```
喫煙記録とAIタバコニュースを1つのアプリで。

━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━

・銘柄ごとに記録を管理
・複数銘柄の登録に対応（名前・1箱の本数・価格を設定）
・今日の喫煙履歴を時系列で表示
・ウィジェットからも素早く記録可能

━━━━━━━━━━━━━━━━━━━━

・銘柄別の内訳を色分けで表示
・合計本数・平均本数・最大本数を確認
・消費金額の集計
・目標ラインの表示
・スワイプで過去のデータを閲覧

・カテゴリ別に閲覧（健康、新商品、業界ニュースなど）
・あなたの喫煙データに基づくおすすめ度を表示
・プルダウンで最新ニュースに更新

・ホーム画面用（小・中サイズ）
・ロック画面用（円形・長方形）
```
