# Smokerアプリ 改善アイディア集

## 現状の整理

現在のオンデバイスLLM活用状況:

- ニュース記事の要約・カテゴリ分類・おすすめ度計算
- リラックスモードの癒しメッセージ生成

現在のデータ資産:

- 喫煙記録（タイムスタンプ、銘柄、金額）
- 銘柄情報（名前、価格、1箱の本数）
- 日/週/月/年の統計データ

---

## カテゴリA: オンデバイスLLM活用アイディア

### A1. AIスモーキングバディ（パーソナルチャット）

ホームまたは専用タブで、喫煙にまつわるカジュアルな会話ができるAIチャット機能。ユーザーの喫煙データを文脈として活用する。

- 「今日の調子はどう？」的な軽い会話
- タバコの豆知識・トリビアを教えてくれる
- 銘柄に関する雑談（「メビウスは元々マイルドセブンって名前だったんだよ」等）
- 喫煙データに基づくポジティブなコメント（「今週は夕方のリラックスタイムが充実してるね」）
- **節煙や禁煙を勧めない**という指示をシステムプロンプトに組み込む

**実装の見込み**: `FoundationModelsService` を拡張し、`LanguageModelSession` のストリーミング応答を活用。会話履歴はセッション内で保持。

---

### A2. AIスモーキングダイアリー（ひとこと日記 + AI洞察）

喫煙記録時にオプションで「ひとこと」（気分、シチュエーション）を添えられる機能。AIがそれを分析して、週末に「今週のまとめ」を生成。

- 記録時に気分タグ（リラックス、仕事の合間、食後、散歩中 等）を選択
- AIが週/月のまとめを生成: 「今週は食後の一服が多かったね。週末はカフェでのリラックスタイムが目立った」
- ポジティブなトーンで自分の喫煙スタイルを振り返れる
- `SmokingRecord` モデルに `mood` / `note` フィールドを追加

---

### A3. AI銘柄ソムリエ

ユーザーの喫煙傾向や好みに基づいて、銘柄に関するパーソナライズされた情報を提供。

- 登録銘柄のフレーバープロファイルの説明をAIが生成
- 「この銘柄を吸っている人は、こんな銘柄も好むかも」的なレコメンド
- コーヒーや飲み物とのペアリング提案（「メンソール系にはブラックコーヒーが合います」等）
- 銘柄ごとの「楽しみ方ガイド」

---

### A4. AIパターンインサイト（ポジティブ分析）

統計画面にAIインサイトカードを追加。データを分析してポジティブな発見を伝える。

- 「あなたのゴールデンタイムは15時台。午後のブレイクを大切にしてますね」
- 「週末は平日より少しゆったりペース。オンオフの切り替えが上手」
- 「今月は3種類の銘柄を楽しんでいます。冒険心がありますね」
- 統計画面の上部にカード形式で1日1つ表示
- データが少ない時期は汎用的なタバコ豆知識を表示

---

### A5. リラックスモードの強化

既存のリラックスモードをAIでさらに進化。

- 時間帯や曜日に応じたメッセージ生成（「金曜の夜、お疲れ様」）
- 短い詩やハイク（俳句）の生成
- ユーザーの喫煙データを反映した「今日の一言」（「50本目の節目、おめでとう」等）
- 呼吸ガイドにAIナレーション

---

## カテゴリB: 一般的な機能改善アイディア

### B1. アチーブメント / 実績システム

ゲーミフィケーション要素で記録を楽しくする。

- 記録系: 「初めての記録」「100本目」「1000本目」「連続30日記録」
- 銘柄系: 「初めての銘柄登録」「5銘柄コレクター」「銘柄マスター」
- 時間系: 「深夜の一服」「早朝の目覚め」「ランチタイムスモーカー」
- 統計系: 「1週間毎日記録」「目標ぴったり達成」
- アチーブメント達成時に祝福アニメーション + ハプティクス

### B4. アプリアイコンクリエイター（ハイブリッド式カスタマイズ）

ユーザーが自分だけのアプリアイコンを簡易的に作成できる機能。自由度を持たせつつも統一感を保つため、パーツ選択+微調整のハイブリッド方式を採用。

**カスタマイズ可能な要素:**

- 煙パーツ: 数種類のベースから選択 → 色・透明度・サイズ・位置を微調整
- タバコイラストパーツ: 数種類のベースから選択 → 色・サイズ・角度・位置を微調整
- 背景色: カラーピッカーまたはプリセットから選択

**適用範囲:**

- 実際のホーム画面アプリアイコン（iOS代替アイコン機能を使用）
- アプリ内表示（起動画面、ヘッダー等）

**技術的なポイント:**

- パーツはSFSymbolsやSwiftUIのShape/Pathで描画（画像アセット最小限）
- 作成したアイコンをレンダリングしてUIImage化 → `UIApplication.shared.setAlternateIconName()` で反映
- カスタマイズ設定はAppSettingsに保存

### B5. その他のカスタマイズ

- テーマカラーの選択
- サウンドエフェクトの追加（記録時の心地よい音）
- 背景エフェクトの種類追加

### B5. カレンダービュー

- 月間カレンダーで喫煙本数をヒートマップ的に表示
- 日ごとの色の濃さで一目で傾向がわかる
- タップで日別詳細表示

## 優先度の提案


| 優先度 | アイディア          | 理由                         |
| --- | -------------- | -------------------------- |
| 高   | A4 AIパターンインサイト | 既存データ+LLMで即実現可能、統計画面の付加価値大 |
| 高   | B1 アチーブメント     | 継続利用のモチベーション、実装が比較的容易      |
| 高   | A2 AIダイアリー     | データの質向上+LLM活用で差別化          |
| 中   | A5 リラックスモード強化  | 既存機能の拡張、実装コスト低             |
| 中   | A1 AIスモーキングバディ | 目玉機能になるがUI設計が必要            |
| 中   | B5 カレンダービュー    | 視覚的に楽しい、実装は中程度             |
| 中   | B4 アイコンクリエイター  | ユニークな差別化機能、愛着が湧く           |
| 低   | A3 AI銘柄ソムリエ    | 面白いが銘柄データの充実が前提            |
| 低   | B5 その他カスタマイズ   | テーマカラー・サウンド等               |
